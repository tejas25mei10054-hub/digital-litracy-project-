## Reflection

For this task, I set up my digital presence across three platforms — GitHub, LinkedIn, and Kaggle.

GitHub was my first priority since it is the most relevant platform for a Cyber Security student. I created my profile README with my name, branch, and a short note about what I hope to learn. I plan to use it to host tools, scripts, and CTF writeups as I progress through my degree.

LinkedIn felt more formal but equally important. I filled in my education details and set up a clean, minimal profile. I know that recruiters and internship coordinators check LinkedIn early, so having it ready from first year itself is a smart move.

Kaggle was new to me. I had heard of it in the context of data science but never used it. After exploring it, I realised it has free Python and machine learning courses that are genuinely useful — even for cyber security, where data analysis is increasingly relevant.

Overall, this task made me realise that building a professional digital identity is not something you do in final year — it starts now.
