## Module Summaries

### Task 1 – Digital Literacy Awareness Infographic (Module 1)
Created a one-page infographic using **Canva** covering what digital literacy is, safe internet practices, and professional online presence. The visual was designed to help batchmates quickly understand key concepts.

### Task 2 – Student Digital Portfolio (Module 2)
Set up professional profiles on **GitHub**, **LinkedIn**, and **Kaggle**. Each platform serves a different purpose — GitHub for code projects, LinkedIn for professional networking, and Kaggle for data science learning.

### Task 3 – Coding & Collaboration Platforms (Module 3)
Completed a beginner challenge on **HackerRank** and created a 5-question **Digital Literacy Awareness Quiz** using Google Forms, linked to a Google Sheet for response collection.

🔗 **Google Form Quiz Link:** *(Add your Google Form link here after creating it)*

### Task 4 – Professional Email & Etiquette Guide (Module 4)
Drafted two professional emails following proper email etiquette, and created a Social Media Do's and Don'ts checklist for responsible online behaviour.

### Task 5 – Cybercrime Awareness: Phishing (Module 5)
Researched and documented a detailed phishing case study, and created a Stay Safe Online checklist with 8+ prevention tips specifically relevant to college students in India, including UPI safety tips.
